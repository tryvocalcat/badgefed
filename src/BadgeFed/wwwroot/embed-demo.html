This page demonstrates how to embed BadgeFed badges on any website. The widget automatically fetches and displays public badges for any recipient.

üöÄ Quick Start - Automatic Method

The easiest way to embed badges. Just include the script and add the HTML container:

<!-- Include the BadgeFed widget script --> <script src="https://your-badgefed-instance.com/js/widget.js"></script> <!-- Add container with data attributes --> <div id="my-badges" data-badgefed-recipient="https://example.com/users/johndoe" data-badgefed-title="John's Achievements" data-badgefed-limit="6"> </div>
Note: Replace https://your-badgefed-instance.com with your actual BadgeFed instance URL and update the recipient URL to match the profile you want to display badges for.

‚ö° JavaScript Method

For more control over when and how the widget loads:

<!-- Include the script --> <script src="https://your-badgefed-instance.com/js/widget.js"></script> <!-- Container --> <div id="custom-badges"></div> <script> // Initialize when ready document.addEventListener('DOMContentLoaded', function() { BadgeFedWidget.render('custom-badges', 'https://example.com/users/johndoe', { title: 'Professional Achievements', limit: 8, showPoweredBy: true }); }); </script>

üîß API Access

Build your own custom badge display using the API endpoints:

// Open Badges JSON-LD format GET /api/embed/badges?recipient=https://example.com/users/johndoe&format=openbadges&limit=10 // Simple JSON format GET /api/embed/badges?recipient=https://example.com/users/johndoe&format=simple&limit=10
